How to use Ledgerly

Private CD ladder and cash-management assistant: track holdings, obligations, and IRA awareness; get decision memos only when something is about to matter.

What Ledgerly does

Ledgerly is a decision-window assistant for safe income: CDs, money market, known bills, and IRA awareness (not investment advice). Your database tables are the source of truth; uploaded documents support and verify.

Ledgerly recommends action only on triggers—a CD maturing soon, a bill due, or an IRA relevant date. Otherwise Home shows All clear.

Quick glossary

TermMeans
AccountA container at one institution. Create accounts first, then add positions inside them.
PositionOne specific holding in an account—e.g. one CD—with amount, rate, and often a maturity date.
PrincipalThe dollar amount in that holding.
APRAnnual interest rate as a number (e.g. 4.5 for 4.5%).
Maturity dateWhen a CD (or similar) matures—often when you can withdraw or renew without penalty.
ObligationSomething you must pay by a date (tax, insurance, loan payment, etc.).
CD ladderStaggered CD rungs shown on Home—institution, rate, maturity, next action.
Decision memoAt maturity: options (hold/roll/MMF/wait), rate comparison, liquidity check.
IRA overviewAwareness-only rows (RMD dates, notes)—not buy/sell advice.
IngestAdd a PDF, image, or pasted text so Ledgerly can search it and answer questions about it.
Doc IDOptional short label for a document (e.g. statement-jan-2024).

Home (start here)

Home loads automatically: status, CD ladder table, IRA awareness lines, upcoming maturities, and bills due.

When a trigger fires, What to do appears with tips plus a Decision memo (rate comparison vs benchmarks, liquidity cross-check, recommendation). Use Refresh to reload.

Add document

Go to Add document, drop one or more PDFs or images on the big zone (or click to browse), then click Ingest. Processing starts right away; you can add more files while earlier ones run—up to 10 per session. When Ledgerly finds a maturity date, a large dialog asks Yes, track this. Check Processing at the bottom of the tab for each file’s result.

  • Upload PDF or images (JPG, PNG). Select several at once in the file picker if you like.
  • The queue is not saved if you close the page—re-select your files to try again.
  • Optional details and folder watching are collapsed—skip them unless you need them.
  • To paste text instead, open Or paste text instead and tap Ingest.

Ask

Preset buttons include Next decision trigger, maturing soon, CD totals, and bills due. Use Speak question or type your own.

After an answer, use Ask follow-up to continue the same conversation. Optional document scope carries forward unless you change the dropdown. New conversation starts fresh.

Manage (advanced)

Data — accounts, positions (ladder fields: start date, next action, liquidity note), obligations, IRA overview.

Documents — review ingested files. Use Delete to remove a document from Ask; any CD or bill tracked from that upload is removed too.

Past advice — history of earlier status checks.

Accounts

Institution-level containers: name (required), optional type and institution.

Positions

Holdings inside an account: account (required), asset type (required), optional description, principal, APR, start date, maturity date, next action, liquidity note.

Obligations

Bills or deadlines: description and due date (required), optional amount estimate and priority.

IRA overview

Awareness only: account name, institution, type, balance estimate, RMD note, next relevant date.

Example workflow (easiest path)

  1. Home: Open the app—your maturities appear automatically.
  2. Add document: Drop files and click Ingest.
  3. Confirm: Tap Yes, track this in the dialog.
  4. Home: Check upcoming maturities anytime; tap Refresh only if needed.

Install & updates

If you received Ledgerly-Portable.zip for Windows, unzip the Ledgerly folder and follow install-instructions.md (Docker Desktop, Setup.bat, desktop shortcut). Developers can follow setup_and_testing.md.

Your data runs on this computer by default (e.g. in Docker). It does not leave your machine unless you change configuration.

When you add a document, Ledgerly automatically tracks CDs and bills it finds. Check Recently added on Home to remove mistakes.
